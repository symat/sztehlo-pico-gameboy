This folder contains definitions of supported devices:

demovga ... DemoVGA board, with output to VGA display 320x240/16-bit
pico ...... simple Raspberry Pico module, extern VGA/DVI display
picotron .. Picotron computer, with output to VGA display with 4-bit color
picoino ... Picoino computer, with output to VGA display 320x240/8-bit and 56-key keyboard.
picopad ... PicoPad console, with output to TFT or VGA display 320x240/16-bit
